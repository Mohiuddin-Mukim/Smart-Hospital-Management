# 🏥 T.Mukimi - Smart Hospital Management System

**T.Mukimi** is a modern, integrated hospital management platform designed to eliminate long waiting times and digitize the healthcare experience. This project prioritizes patient convenience and operational transparency through real-time data tracking.

---

## ✨ Core & Unique Features

### 🔴 Live Token & Queue Tracking (Unique Feature)
- **Real-time Serial Monitoring:** Patients can see exactly which token number is currently inside the doctor's room.
- **Live Updates:** The dashboard shows the current patient being served and the upcoming queue.
- **Time Optimization:** By tracking their serial live from their phones, patients can utilize their time elsewhere instead of waiting in crowded hospital halls.

### 📅 Simplified Appointment System
- Intuitive interface for browsing doctors' schedules and booking slots instantly.
- Automated token generation upon successful booking.

### 📜 Digital Prescription & Medical History
- Secure storage for all past prescriptions.
- Patients can access their medical history and previous doctor recommendations at any time.

---

## 🛠️ Tech Stack & Philosophy

I am developing the backend of this project **without the help of AI** to master the core fundamentals and architectural patterns of Spring Boot. Every logic gate and database schema is hand-coded to ensure a deep understanding of full-stack development.

- **Backend:** Java, Spring Boot 4.x, Spring Data JPA, Spring Security.
- **Frontend:** Tailwind CSS, JavaScript, HTML5.
- **Database:** MySQL.
- **Real-time Updates:** WebSockets for live token synchronization.
- **Tools:** Maven, IntelliJ IDEA, Postman.
